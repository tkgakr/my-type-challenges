# 学習環境の起動
```bash
# VSCodeでプロジェクトを開く
cd my-type-challenges
code .

# Dev Containerを起動
# Cmd/Ctrl + Shift + P → "Dev Containers: Reopen in Container"

# 起動時に最新の original-type-challenges を取得します
```

# 学習フロー
1. チャレンジを開始
```bash
npm run solve 4 easy
```
2. ソリューションを編集（IDE上で）
3. 進捗を更新
```bash
npm run progress
```
